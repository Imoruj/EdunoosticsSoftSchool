# Sales Demo Script

## Goal

Show that Edunostics is not just a report card generator. It is a full academic workflow system for Nigerian schools.

## Opening

Thank you for joining. I will show how Edunostics helps a school manage the result cycle from setup to score entry, review, broadsheet, report card generation, attendance, and academic tools.

The main problem we solve is the stress of running school results through scattered spreadsheets and manual checks.

## Common Objections

### We already use Excel.

Excel can work for a small workflow, but it becomes risky when many teachers, classes, approvals, report templates, broadsheets, comments, attendance records, and publishing steps are involved. Edunostics gives the school a controlled workflow.

### Will it match our report card format?

The platform supports school branding, logos, signatures, grading rules, attendance, academic tables, behaviour ratings, psychomotor skills, and comments. The template can be configured to fit the school.

### Is it only for report cards?

No. It includes report cards, broadsheets, score entry, attendance, lessons, quizzes, assignments, scheme of work, class progress, role access, and school settings.

### Can teachers use it?

Yes. Teachers can enter scores, manage class work, create assignments and quizzes, and work within the permissions assigned by the school.
