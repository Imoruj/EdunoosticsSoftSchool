# Edunostics Marketing Kit

This folder contains the first campaign kit for promoting Edunostics using the real app workflows and screenshots.

## Files

- `campaign-plan.md` - positioning, audiences, funnel, campaign offer, rollout, and metrics.
- `landing-page-copy.md` - landing page copy structured by section.
- `ad-copy.md` - Facebook, Instagram, Google Search, and LinkedIn copy.
- `whatsapp-scripts.md` - WhatsApp broadcast and follow-up scripts.
- `demo-script.md` - sales demo flow and objection handling.
- `creative-brief.md` - screenshot creative directions and image generation prompts.
- `social-calendar.md` - 14-day campaign posting plan.
- `prospectus-book.md` - complete long-form prospectus book for the app.

## Generated PDFs

- `output/pdf/edunostics-prospectus-book.pdf` - long-form image-rich prospectus.
- `output/pdf/edunostics-visual-marketing-prospectus.pdf` - shorter visual marketing prospectus with screenshots, callouts, charts, shapes, and annotations.
- `output/pdf/edunostics-client-product-prospectus.pdf` - client-facing product prospectus with polished wording for interested schools.
- `output/pdf/edunostics-premium-product-prospectus.pdf` - preferred world-class client prospectus with matched feature pages, product screenshots, annotations, and concise buyer-facing copy.

## Recommended Execution Order

1. Build or update the landing page using `landing-page-copy.md`.
2. Prepare screenshot creatives using `creative-brief.md`.
3. Start WhatsApp outreach using `whatsapp-scripts.md`.
4. Launch small paid tests using `ad-copy.md`.
5. Run demos using `demo-script.md`.
6. Post daily using `social-calendar.md`.
